Creates a 2D OpenGL texture from raw image data. Note that the raw data is NOT freed after the upload (so the user can load various versions).

Parameters

heightthe pointer of the height of the image in pixels ( if the texture size change, height will be overrided with the new height )
channelsthe number of channels: 1-luminous, 2-luminous/alpha, 3-RGB, 4-RGBA
reuse_texture_ID0-generate a new texture ID, otherwise reuse the texture ID (overwriting the old texture)

Parameters

buffer_lengththe size of the buffer in bytes
face_orderthe order of the faces in the file, any combination of NSWEUD, for North, South, Up, etc.
force_channels0-image format, 1-luminous, 2-luminous/alpha, 3-RGB, 4-RGBA
reuse_texture_ID0-generate a new texture ID, otherwise reuse the texture ID (overwriting the old texture)
flagscan be any of SOIL_FLAG_POWER_OF_TWO | SOIL_FLAG_MIPMAPS | SOIL_FLAG_TEXTURE_REPEATS | SOIL_FLAG_MULTIPLY_ALPHA | SOIL_FLAG_INVERT_Y | SOIL_FLAG_COMPRESS_TO_DXT | SOIL_FLAG_DDS_LOAD_DIRECT
